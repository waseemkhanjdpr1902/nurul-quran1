Indices: Real-time tracking and tips for Nifty 50 and Bank Nifty.

Signal Cards: Professional cards showing: Asset Name, Entry Price, Stop Loss (SL), Target 1, Target 2, and "Risk/Reward Ratio."

Social Broadcast Engine: >    * WhatsApp Integration: A "Share to WhatsApp" button that formats the signal into a clean, professional text message.

Telegram Bot: A "Post to Channel" button that uses the Telegram Bot API to automatically broadcast the tip to a specific Group/Channel.

Frontend: React with Tailwind CSS (Dark Mode optimized for traders).

Dashboard: High-contrast "TradingView" style aesthetic (Dark backgrounds, Neon Green for 'Buy', Neon Red for 'Sell').

Notification Center: Push notifications for "Target Hit" or "Stop Loss Hit" alerts.

Stripe Integration: Monthly subscription for "Premium Pro" signals.

Disclaimer: A mandatory scrolling footer and pop-up disclaimer regarding financial risk and SEBI/Regulatory compliance.

Instructions: Scaffold the app, set up the Supabase schema for signals, and write the Node.js logic for the Telegram Bot integration.

How to Make the "WhatsApp & Telegram" Parts Work
The AI will write the code, but you need to provide the "pipes" for the messages to travel through:

For Telegram:

Search for @BotFather on Telegram.

Create a new bot and get the API Token.

Add the bot as an Admin to your Telegram Channel.

Give the Token to your AI, and it will handle the rest.

For WhatsApp:

The Simple Way: The AI uses a whatsapp://send?text= link. This opens the user's WhatsApp and pastes the tip automatically.

The Pro Way: Use the Twilio WhatsApp API if you want the app itself to send the message automatically to thousands of people without you opening your phone.

Strategic Advice for Deployment
Data Accuracy: Since you mentioned Nifty and Bank Nifty, ensure you use a reliable data provider. In India, many traders use TrueData or Global Data Feeds via API for the most accurate NSE/BSE prices.

The "Disclaimer" is Mandatory: Financial apps are heavily regulated. Ensure your app has a clear section stating that "Tips are for educational purposes only."

Deployment: Use the PWABuilder method I mentioned earlier to turn this into an Android App. Traders prefer apps that stay open in the background with "Push Notifications" so they don't miss an entry price.
