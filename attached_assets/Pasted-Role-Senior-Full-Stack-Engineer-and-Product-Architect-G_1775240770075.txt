Role: Senior Full-Stack Engineer and Product Architect.

Goal: Build a fully functional PWA (Progressive Web App) called "StockSwap B2B." This app allows local retail shop owners to list unsold/slow-moving inventory for other businesses to buy at a discount.

1. Core Technical Stack:

Frontend: React + Tailwind CSS (Mobile-first design).

Backend: Node.js (Express).

Database: Supabase (PostgreSQL) for user profiles, product listings, and chat.

Maps Integration: Use Leaflet.js (OpenStreetMap) to show a "Heatmap" of available stock within a 5km–10km radius.

2. High-Priority Features (FULLY FUNCTIONAL):

The "Fast List" Camera: A camera interface where a user takes a photo. The AI (use a lightweight vision library or Google Vision API) must suggest the category and brand.

Hyper-Local Discovery: A dashboard showing "Deals Near Me." Listings must show Distance (km), Original Price, Discounted Price, and Expiry Date (if applicable).

B2B Chat: A simple real-time chat system using Supabase Realtime so owners can negotiate a bulk price.

Verification System: A badge for "Verified Shops" to prevent scam listings.

3. Monetization & AdMob:

AdMob Integration: A 320x50 banner at the bottom of the "Search" and "My Listings" screens. Use ADMOB_BANNER_ID from secrets.

Featured Listings: Integrate Stripe to allow users to pay $1 to "Boost" their listing to the top of the local feed for 24 hours.

4. Technical Implementation Flow:

Set up the Supabase schema first: Users, Shops, Listings (with Latitude/Longitude), and Messages.

Create the "Seller Journey": Upload Photo -> Fill Details -> Set Location -> Publish.

Create the "Buyer Journey": Search by Category -> Filter by Distance -> Start Chat.

Implement the "Terms of Service" popup stating: "This is a B2B exchange. All inspections must be done in person at the shop location."

Instructions: Build the entire scaffold. Focus on the mobile UI being extremely simple for non-tech-savvy shop owners. Ensure the AdMob and Stripe components are ready for live keys.

Why this scales to $100M
Network Effect: The more shops that join in a specific city (like Mumbai or Delhi), the more valuable the app becomes.

The "Data Goldmine": You will eventually know exactly which brands are failing in which neighborhoods. Big companies (Unilever, P&G, Samsung) will pay millions to see your "Dead Stock Trends" report to adjust their own manufacturing.

Logistics Expansion: Once the marketplace is busy, you can add a "Book a Delivery" button. You can partner with local logistics startups (like Dunzo or Porter) and take a commission on every delivery.
