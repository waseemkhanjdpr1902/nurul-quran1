🔐 PART 1 — Add All API Keys & Secret Configuration
Set up a secure configuration system for all API keys and secrets used in this project.
1. Create a .env file in the root directory with these keys:
env# Quran & Islamic APIs (these are free, no key needed but add base URLs)

3. Add Replit Secrets (Environment Variables) in Replit:

Go to Replit → Secrets (🔒 icon) in the left sidebar
Add each key from the .env file as a separate secret
In your server-side code (if using Node/Express), access them via process.env.KEY_NAME
Never hardcode any API key directly in HTML or JS files

4. Create a .gitignore file to protect secrets:

🚀 PART 2 — Deploy to Existing GitHub Repository nurul-quran
Step 1 — Initialize Git in Replit and connect to GitHub
Run these commands in the Replit Shell:
bash# Set your Git identity
git config --global user.email "your-github-email@gmail.com"
git config --global user.name "YourGitHubUsername"

# Initialize git if not already done
git init

# Connect to your existing GitHub repo
git remote add origin https://github.com/YourGitHubUsername/nurul-quran.git

# Pull existing repo content first to avoid conflicts
git pull origin main --allow-unrelated-histories
Step 2 — Stage, Commit and Push all files
bash# Add all files except secrets
git add .

# Commit with message
git commit -m "🕌 Nurul Quran - Full Islamic Education Web App with all features"

# Push to main branch
git push -u origin main
Step 3 — Set up GitHub Secrets for CI/CD

Go to your GitHub repo → Settings → Secrets and Variables → Actions
Click "New repository secret" and add each key:

Secret NameValueOPENWEATHER_API_KEYyour key hereGOOGLE_MAPS_API_KEYyour key hereEMAILJS_PUBLIC_KEYyour key hereFIREBASE_API_KEYyour key here
Step 4 — Enable GitHub Pages for live deployment
Create this file: .github/workflows/deploy.yml
yamlname: Deploy Nurul Quran to GitHub Pages

on:
  push:
    branches:
      - main

jobs:
  deploy:
    runs-on: ubuntu-latest
    steps:
      - name: Checkout code
        uses: actions/checkout@v3

      - name: Deploy to GitHub Pages
        uses: peaceiris/actions-gh-pages@v3
        with:
          github_token: ${{ secrets.GITHUB_TOKEN }}
          publish_dir: ./
Step 5 — Activate GitHub Pages

Go to your GitHub repo → Settings → Pages
Under Source, select Branch: gh-pages → / (root)
Click Save
Your site will be live at:
https://YourGitHubUsername.github.io/nurul-quran
